## Features
* Responsive
* Infinite Scrolling
* Filters (JobRole,Location,minExp,companyName , Remote/inOffice)

## How to run the projects

* Make sure NodeJS is installed on your sysytem

```
  git clone https://github.com/AzadTom/JobPlatform-Assignment.git
  npm install
  npm run dev

```
